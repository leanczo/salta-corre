# Salta Corre

Calendario de carreras de running en la provincia de Salta, Argentina - Temporada 2026.

## Carreras incluidas

| Carrera | Fecha | Tipo | Distancias |
|---------|-------|------|------------|
| Maratón de las Luces | 28 Feb | Nocturna | 3K, 7K, 16K |
| NB Race Series Salta | 29 Mar | Ruta | 10K, 21K |
| Carrera Nocturna Salta (FC Running) | 25 Abr | Nocturna | 4K, 7K, 14K, 21K |
| Running Trip | 18-20 Jun | Trail | Vertical, 5K, 10K, 21K, 35K |
| Maratón HIRPACE | Sep/Oct | Solidaria | 1.5K, 3K, 10K |
| Qhapaqñam Trail Running | Oct | Trail | 9K, 15K, 22K, 42K, Vertical, 3K Kids |
| La Caldera Team Trail | Nov | Trail | 5K, 15K, 30K |
| Triatlón de la Cumbre | Nov | Triatlón | Natación + Ciclismo + Trepada |
| Maratón San Silvestre | 31 Dic | Ruta | Participativa |

## Funcionalidades

- Filtros por tipo de carrera (ruta, trail, nocturna, solidaria, triatlón)
- Marcar carreras como "inscripto" (se guarda en localStorage)
- Cuenta regresiva para la próxima carrera
- Calendario interactivo con navegación mensual
- Tema claro/oscuro (detecta preferencia del sistema)
- Diseño mobile-first

## Tech stack

- React 19
- Vite
- Deploy en Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy en Vercel

1. Conectar el repositorio de GitHub en [vercel.com](https://vercel.com)
2. Framework preset: **Vite**
3. Build command: `npm run build`
4. Output directory: `dist`

Vercel detecta Vite automáticamente, no necesita configuración extra.
